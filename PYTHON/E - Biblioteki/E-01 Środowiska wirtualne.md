.venv to katalog środowiska wirtualnego
W PyCharm'ie tworzy się automatycznie przy tworzeniu nowego projektu

`.\.venv\Scripts\activate`					- inicjalizowanie środowiska wirtualnego
`.\.venv\Scripts\Activate.ps1`
`deactivate`											- deaktywacja środowiska wirtualnego
`pip freeze > requirements.txt`		- generowanie requirements.txt
`pip install -r requirements.txt`	- instalowanie pakietów z requirements.txt

## Pakiety
Py.P.I. - https://pypi.org/ Python Package Index
PSL - Python Standard Library
BIF-y - Praktyczne, potężne i ogólne funkcje wbudowane

`pip list` - pozwala zobaczyć jakie pakiety są zainstalowane w środowisku
