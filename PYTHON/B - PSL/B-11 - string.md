`import string

### Najważniejsze stałe w `string`

- `string.ascii_lowercase` → `'abcdefghijklmnopqrstuvwxyz'`
- `string.ascii_uppercase` → `'ABCDEFGHIJKLMNOPQRSTUVWXYZ'`
- `string.ascii_letters` → małe + duże litery razem
- `string.digits` → `'0123456789'`
- `string.hexdigits` → `'0123456789abcdefABCDEF'`
- `string.octdigits` → `'01234567'`
- `string.punctuation` → wszystkie znaki interpunkcyjne (`!"#$%&'()*+,-./:;<=>?@[\]^_{|}~`)
- `string.printable` → znaki, które da się normalnie wydrukować (litery, cyfry, spacje, interpunkcja)
- `string.whitespace` → wszystkie białe znaki (`' \t\n\r\x0b\x0c'` → czyli spacja, tab, nowa linia itd.)
