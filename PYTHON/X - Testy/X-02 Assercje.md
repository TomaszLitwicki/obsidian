## Często używane asercje

| Metoda                | Sprawdza                   |
| --------------------- | -------------------------- |
| `assertTrue(x)`       | czy wyrażenie jest True    |
| `assertFalse(x)`      | czy jest False             |
| `assertIn(a, b)`      | czy `a` znajduje się w `b` |
| `assertIsNone(x)`     | czy `x is None`            |
| `assertGreater(a, b)` | czy `a > b`                |
| `assertEqual(a, b)`   | czy a == b                 |
