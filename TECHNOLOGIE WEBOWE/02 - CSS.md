selector - wybrany element na stronie (na przykład h1 albo jakieś id `<div>`'a ale wtedy potrzebny jest # lub .)
property - właściwość
value - wartość tej właściwości

Linki (hiperłącza)
Właściwości linków - 3 stany (nieodwiedzony/odwiedzony/najechany)
 - a:link {właściwości} - link nieodwiedzony
 - a:visited {właściwości} - link odwiedzony
 - a:hover {właściwości} - hover over (po najechaniu)


\# - selector dla konkretnego `id="nazwa"` - *może być tylko jedno*
. - selector dla całej klasy `class="nazwa'` - *może być wiele elementów*
