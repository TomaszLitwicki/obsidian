program **less** gdy wynik jest dłuższy niż jedno okno terminala i umożliwia przewijanie i inne:

| Klawisz   | Co robi              |
| --------- | -------------------- |
| `q`       | wyjście              |
| `↑` / `↓` | przewijanie          |
| `space`   | strona w dół         |
| `b`       | strona w górę        |
| `/tekst`  | wyszukiwanie         |
| `n`       | następne dopasowanie |
