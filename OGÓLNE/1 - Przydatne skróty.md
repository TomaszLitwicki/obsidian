# PyCharm:

| SKRÓT          | OPIS                                               |
| -------------- | -------------------------------------------------- |
| Ctrl + /       | Wstawianie # do wszystkich zaznaczonych lini kodu  |
| Shift + F6     | Zmienia nazwę wszystkich zmiennych (takich samych) |
| Ctrl + Alt + L | Reformatowanie kodu                                |


# IntelliSense

| SKRÓT                 | OPIS                                            |
| --------------------- | ----------------------------------------------- |
| Ctrl + Spacja         | podstawowe podpowiedzi                          |
| Ctrl + Shift + Spacja | inteligentne podpowiedzi                        |
| Ctrl + P              | podpowiedzi parametrów w wywołaniu funkcji      |
| Ctrl + Q              | szybka dokumentacja (opis klasy, funkcji, itd.) |
